# Webová aplikace pro výpočet a 
### Instalace (Je nutné mít Xampp kvůli Apache)
1. Naklonujte si repozitář s projektem na svůj disk
- `git clone https://github.com/PanVat/cerne-diry.git`
2. Přesuňte se do složky webu
- `cd C:\xampp\htdocs`
3. Vytvořte virtuálního prostředí do složky .venv
- `python -m venv .venv`
4. Aktivujte virtuální prostředí 
- `.venv\Scripts\activate` (ve Windows)
5. Nainstalujte balíčky a závislosti
- `pip install -r requirements.txt`
6. Zapněte Xampp a spusťte Apache
7. Do prohlížeče zadejte
8. `http://localhost`
